HTML + CSS Cheat Sheet (cập nhật 2025)

1. Cấu trúc HTML cơ bản

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8"/>
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <title>Trang của tôi</title>
  <link rel="stylesheet" href="style.css"/>
</head>
<body>

  <header>…</header>
  <main>…</main>
  <footer>…</footer>

</body>
</html>

2. Các thẻ HTML quan trọng nhất (semantic)

ThẻMục đíchVí dụ
<header>Phần đầu trang / section headerlogo + nav
<nav>Menu điều hướngnavbar

<figure> + <figcaption>Ảnh + chú thích—

3. CSS Cơ bản – Cách viết & Quy tắc

/* 1. Selector cơ bản */
p { color: navy; }
.btn-primary { ... }
#sidebar { ... }

/* 2. Combinators */
article p          /* descendant */
nav > ul           /* child */
h2 + p             /* adjacent sibling */
li ~ li            /* general sibling */

/* 3. Pseudo-classes phổ biến */
a:hover, a:focus
button:active
input:focus
li:nth-child(odd)
tr:nth-of-type(2n+1)

/* 4. Pseudo-elements */
p::first-line
h1::before { content: "★ "; }
::marker { color: coral; }

4. Flexbox – Cheat Sheet nhanh

.container {
  display: flex;
  flex-direction: row | column | row-reverse;
  flex-wrap: nowrap | wrap | wrap-reverse;
  justify-content: flex-start | center | space-between | space-evenly;
  align-items: stretch | center | flex-start | baseline;
  gap: 1.5rem;               /* row + column gap */
  row-gap: 20px;
  column-gap: 40px;
}

.item {
  flex: 1 1 300px;          /* grow shrink basis */
  align-self: center;
  order: -1;
}

5. CSS Grid – Cheat Sheet nhanh

.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header header"
    "sidebar main   ads   "
    "footer footer footer";
  gap: 1.5rem;
}

.item-header    { grid-area: header; }
.card-featured  { grid-column: span 2; }
.hero           { grid-column: 1 / -1; }

7. Modern CSS hay dùng 2025

/* Màu tương đối */
background: oklch(from var(--primary) l c h / 0.15);

/* Container queries */
@container (min-width: 500px) {
  .card { flex-direction: row; }
}

/* Scroll snap */
.scroll-container {
  scroll-snap-type: x mandatory;
}
.item { scroll-snap-align: center; }

/* Text wrap pretty */
text-wrap: pretty;          /* hoặc balance */

/* Truncate đa dòng */
display: -webkit-box;
-webkit-box-orient: vertical;
-webkit-line-clamp: 3;
overflow: hidden;

8. Responsive Design – Media Queries hiện đại

@media (width <= 480px)  { ... }   /* mobile */
@media (width <= 768px)  { ... }   /* tablet */
@media (width >= 1024px) { ... }   /* desktop */

/* Cách viết gọn & phổ biến 2025 */
@container style(--theme: dark) { ... }

@supports (gap: 1rem) and (aspect-ratio: 16/9) {
  /* code cho trình duyệt hỗ trợ */
}
